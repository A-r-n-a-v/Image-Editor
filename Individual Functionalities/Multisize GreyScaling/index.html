Converting an Image to GreyScale

We can not add more pixels to an image, we can only make the GreyScaling faster by reducing the number of pixels in the image. Different standard sizes can be selected for GreyScaing. Remember, if the original size of image is less than selected GreyScaling size, image will be GreyScaled according to the original size. If nothing matches one's needs, custom sizes can also be provided by the user.
